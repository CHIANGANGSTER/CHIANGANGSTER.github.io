视角转换微调工作流

QWEN IMAGE EDIT · MULTIANGLE CAMERA · SEEDVR2

这个工作流用 Qwen Image Edit 做画面重绘，用多角度相机控制 LoRA 调整视角，再用 SeedVR2 放大结果。 你只改三个相机参数：水平角度、垂直角度、镜头距离。它适合轻量视角微调，不适合大幅重建主体。

工作流定位

Small camera move, not a full redesign

你把原图交给工作流，让它在保持主体和画面关系的前提下换一个观察角度。工作流不负责重新设计产品、替换材质或生成全新场景。它更像一次相机微调：换一点水平角度，抬低一点视线，或者拉近镜头。

核心模型

主模型使用 Qwen Image Edit。工作流用图像编辑链路读取原图，再按相机控制参数生成新角度。

控制 LoRA

多角度相机控制 LoRA 负责相机运动。Lightning LoRA 把采样压到 4 steps，适合快速预览。

后处理

SeedVR2 负责放大。工作流先把结果送进放大节点，再保存输出图，减少低分辨率预览带来的误判。

三个相机参数

Start small, then compare

horizontal_angle

调水平角度。范围可到 360 度。先从小幅度开始，例如 15、30、45，再看主体结构是否稳定。

vertical_angle

调垂直角度。范围约 60 度。你用它控制俯视、平视、仰视，不要一次拉满。

zoom

调镜头距离。0 接近远景，10 接近特写。产品图建议先用中间值，避免裁掉关键轮廓。

使用顺序

One input image, one controlled camera move

放入原图

先放入清晰原图。主体边缘、透视线和材质反光越清楚，视角微调越稳。

先调水平角度

先只改 horizontal_angle。确认主体没有变形后，再调整 vertical_angle。

最后调镜头距离

最后调 zoom。镜头距离会影响裁切，先保存中景版本再做特写。

结构漂移就回退

如果主体结构开始漂移，退回上一个参数值。不要用提示词硬拉回结构。
